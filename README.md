# MaiUp!!!

一个使用 `Kotlin Multiplatform` 构建的 `舞萌 | 中二` 传分器，支持 `落雪 | 水鱼` 查分器。

## 功能 RoadMap

- [x] 更换主题
- [ ] 上传成绩
- [x] 查询歌曲
- [x] 查询 B50
- [x] 支持落雪 OAuth
- [ ] 其他功能 ...

## 构建

### Android

通过 IDE 构建或通过终端构建

- macOS/Linux
  ```shell
  ./gradlew :composeApp:assembleDebug
  ```
- Windows
  ```shell
  .\gradlew.bat :composeApp:assembleDebug
  ```

### Desktop (JVM)

通过 IDE 构建或通过终端构建

- macOS/Linux
  ```shell
  ./gradlew :composeApp:run
  ```
- Windows
  ```shell
  .\gradlew.bat :composeApp:run
  ```
